    It was a lot of fun making this website! At first, I was going to do a tutorial using React to create my website. I decided to go backwards and learn the basics first before trying to make a more complicated React app. I expected to do a few JS tutorials first, but later realised that I could make a decent looking website without React, so I just added more parts to the first tutorial to create this website.
